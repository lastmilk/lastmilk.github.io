关于CMCL API

概要

CMCL API是@海晶鱼（QQ:2735209779）开发，CraftBBS（craftbbs.com）推广的一款minecraft游戏本体下载镜像站。与MCBBS源和BMCLAPI类似。

CMCLAPI当前正处于公测阶段，目前开放Minecraft 游戏本体下载以及 assest游戏文件下载。

食用

         版本信息

        http://launchermeta.mojang.com/mc/game/version_manifest.json -> https://res.craftbbs.xyz/version_manifest.json

        http://launchermeta.mojang.com/mc/game/version_manifest_v2.json -> https://res.craftbbs.xyz/version_manifest_v2.json

         版本和版本JSON以及AssetsIndex

        将版本信息内的URL中的https://launchermeta.mojang.com/ 和 https://launcher.mojang.com/ 替换为 https://res.craftbbs.xyz

         Assets

        http://resources.download.minecraft.net -> https://res.craftbbs.xyz/assest/

         Libraries

        https://libraries.minecraft.net/ -> https://res.craftbbs.xyz/

         Mojang java

        https://launchermeta.mojang.com/v1/products/java-runtime/2ec0cc96c44e5a76b9c8b7c39df7210883d12871/all.json -> https://res.craftbbs.xyz/all.json

         Forge

        https://files.minecraftforge.net/maven -> 当前开发中

         Liteloader

        http://dl.liteloader.com/versions/versions.json -> 当前开发中

         Optifine

        请遵照API，无官方结构

         authlib-injector

        当前开发中

         fabric

        当前开发中

         quilt (由于上游API存在bug，暂时不可用)
