关于CMCL API

CMCL API是@海晶鱼（QQ:2735209779）开发，CraftBBS（craftbbs.com）推广的一款Minecraft游戏本体下载镜像站。与MCBBS源和BMCLAPI类似。

CMCLAPI当前正处于公测阶段，目前开放Minecraft游戏本体下载以及assets游戏文件下载。

赞助

二维码是自愿赞助，基本每人每月捐2元CMCL就能维持下去。如果有额外费用，将会用于搭建更多的节点。就算不捐，站长也会自费维护下载源。
